# SYNOPSIS

    use Map::Metro;

# DESCRIPTION

# Status

This map includes:

- All three current lines (as of 2019-10-29) of the Copenhagen metro ([wikipedia](https://en.wikipedia.org/wiki/Copenhagen_metro))
